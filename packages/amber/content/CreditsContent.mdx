- Token Based Authentication for open source projects from [Auth0](https://auth0.com/?utm_source=oss&utm_medium=gp&utm_campaign=oss).
- The table control used throughout this site is [React-Table](https://react-table.tanstack.com/), all GraphQL client access uses
  [React-Query](https://react-query.tanstack.com/), both by Tanner Linsley.
- The UI toolkit is [Material UI](https://material-ui.com/), with parts inspired by [Material Kit React](https://demos.creative-tim.com/material-kit-react)
- All the code is written in [TypeScript](https://www.typescriptlang.org/)
- Database from [PostgreSQL](https://www.postgresql.org)
- Automatic Postgres to GraphQL from [PostGraphile](https://www.graphile.org/postgraphile/)
- GraphQL schema to React-Query from [Graphql CodeGen](https://graphql-code-generator.com/)
- GraphiQL Explorer by [OneGraph](https://www.onegraph.com/)
- Schema migration tools by [Knex.js](http://knexjs.org/)
- Forms using [Formik](https://formik.org/)
- Email sent using [Nodemailer](https://nodemailer.com/about/')
- And of course, the whole UI is built using [React](https://reactjs.org/)
- Various useful utilities from [Use Hooks](https://usehooks.com)
- The Old Stamper font used in the "virtual" convention overlay is from [https://www.dafont.com/typesgal.d3983](https://www.dafont.com/typesgal.d3983)
